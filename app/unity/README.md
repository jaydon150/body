# app/unity — reserved

This directory is a placeholder for a future native/VR client built in Unity. It is **out of scope for v1** per `docs/orchestrator/master-spec.md` §2 (stack) and §7 (phasing).

If Unity is activated in a later phase, an ADR will document the decision and this directory will receive the Unity project skeleton.

For v1, the web viewer at `../web/` is the only target.
